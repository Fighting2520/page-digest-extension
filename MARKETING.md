## 📝 Store Listing Copy

### Short Description (132 characters max)
```
AI-powered webpage summarization. Extract insights, translate, and save summaries with one click. Powered by Google Gemini.
```

### Detailed Description

```
Transform any webpage into concise, actionable insights with QuickRead AI.

🚀 KEY FEATURES

📝 Smart Summarization
Get instant, intelligent summaries of any article, blog post, or documentation page. Save hours of reading time.

📊 Data Extraction
Automatically extract key statistics, facts, and figures from complex content.

🎯 Key Points
Generate bullet-point highlights for quick scanning and easy reference.

🌐 Smart Translation
Auto-detect language and translate content between English and Chinese.

📚 History
Save and revisit your recent summaries. Never lose important insights.

⚡ Keyboard Shortcuts
Work faster with customizable hotkeys:
• Alt+S - Quick summarize
• Alt+Q - Open Q&A mode
• Alt+T - Translate page
• Alt+D - Toggle side panel

🎨 Modern Interface
Clean, professional design inspired by Notion and Linear. Works seamlessly in side panel or popup mode.

🔒 Privacy First
• No data collection
• All data stays local on your device
• Direct API calls to Google Gemini
• Open source and transparent

💡 HOW IT WORKS

1. Install the extension
2. Get a free Google Gemini API key (takes 2 minutes)
3. Click the QuickRead AI icon on any webpage
4. Choose your action: Summarize, Extract Data, Key Points, or Translate
5. Get instant AI-powered insights

✨ PERFECT FOR

• Researchers and students
• Content creators and writers
• Busy professionals
• Anyone who reads a lot online

🆓 FREE TO USE

QuickRead AI uses Google's free Gemini API. No subscription required.

📖 OPEN SOURCE

Full source code available on GitHub. Audit the code yourself and contribute improvements.

---

Powered by Google Gemini 2.5 Flash - the latest AI model for fast, accurate text understanding.

Questions? Visit our GitHub repository or contact support.
```
