# Changelog

All notable changes to QuickRead AI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-03-09

### 🎉 Major Release - Brand Refresh

#### Changed
- **Rebranded** from PageDigest to **QuickRead AI**
- Updated extension name and description for better discoverability
- Improved manifest metadata for Chrome Web Store readiness

#### Added
- **Privacy Policy** - Comprehensive privacy documentation (PRIVACY.md)
- **Enhanced README** - Professional documentation with clear installation steps
- **Changelog** - Version tracking and release notes
- Side panel mode for persistent access
- Keyboard shortcuts (Alt+S, Alt+Q, Alt+T, Alt+D)
- History tracking for recent summaries
- Custom prompt support
- Smart translation (auto-detect language)
- Data extraction mode
- Key points generation
- Markdown rendering support

#### Fixed
- Improved error handling for API failures
- Better content extraction from complex webpages
- UI responsiveness on different screen sizes

#### Security
- All data stored locally (no external servers)
- Direct API calls to Google Gemini (no intermediary)
- Secure API key storage using Chrome's encrypted storage
