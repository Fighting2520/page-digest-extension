# Privacy Policy for QuickRead AI

**Last Updated:** March 9, 2025

## Overview

QuickRead AI ("we", "our", or "the extension") is committed to protecting your privacy. This policy explains how we handle your data.

## Data Collection

**We DO NOT collect, store, or transmit any personal data to our servers.**

### What Data is Processed

- **Webpage Content**: When you use the summarization feature, the text content of the current webpage is sent directly to Google's Gemini API for processing.
- **API Key**: Your Google Gemini API key is stored locally in your browser's Chrome storage.
- **Summary History**: Your summary history is stored locally on your device.

### What We DO NOT Collect

- Personal information (name, email, address)
- Browsing history
- Cookies or tracking data
- Any data beyond what's necessary for the extension to function

## Third-Party Services

### Google Gemini API

QuickRead AI uses Google's Gemini API to generate summaries. When you use the extension:

- Webpage content is sent directly from your browser to Google's servers
- Google's privacy policy applies to this data: https://policies.google.com/privacy
- We do not intercept, store, or access this data

### Your API Key

- Your Gemini API key is stored locally using Chrome's storage API
- It is never transmitted to any server other than Google's Gemini API
- You can delete it at any time through the extension settings

## Data Storage

All data is stored locally on your device:

- **Chrome Storage**: API key and user preferences
- **Local Storage**: Summary history

You can clear this data at any time by:
1. Removing the extension
2. Clearing Chrome's extension data
3. Using the "Clear History" button in the extension

## Permissions

QuickRead AI requests the following permissions:

- **activeTab**: To read the content of the current webpage
- **storage**: To save your API key and preferences locally
- **scripting**: To inject content scripts for text extraction
- **sidePanel**: To display the side panel interface
- **contextMenus**: To add right-click menu options

These permissions are used solely for the extension's functionality and not for data collection.

## Data Security

- All communication with Google Gemini API uses HTTPS encryption
- Your API key is stored securely in Chrome's encrypted storage
- No data is transmitted to third-party servers

## Children's Privacy

QuickRead AI does not knowingly collect data from children under 13. The extension is intended for general audiences.

## Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated "Last Updated" date.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository:
https://github.com/Fighting2520/page-digest-extension

## Your Rights

You have the right to:
- Access your locally stored data
- Delete your data at any time
- Stop using the extension

# 隐私政策 - QuickRead AI

**最后更新：** 2025年3月9日

## 概述

QuickRead AI（"我们"或"本扩展"）致力于保护您的隐私。本政策说明我们如何处理您的数据。

## 数据收集

**我们不会收集、存储或传输任何个人数据到我们的服务器。**

### 处理的数据

- **网页内容**：当您使用摘要功能时，当前网页的文本内容会直接发送到 Google Gemini API 进行处理。
- **API 密钥**：您的 Google Gemini API 密钥存储在浏览器的本地存储中。
- **摘要历史**：您的摘要历史记录存储在您的设备本地。

### 我们不收集的内容

- 个人信息（姓名、邮箱、地址）
- 浏览历史
- Cookie 或跟踪数据
- 任何超出扩展功能所需的数据

## 第三方服务

### Google Gemini API

QuickRead AI 使用 Google Gemini API 生成摘要。当您使用扩展时：

- 网页内容直接从您的浏览器发送到 Google 服务器
- Google 的隐私政策适用于这些数据：https://policies.google.com/privacy
- 我们不会拦截、存储或访问这些数据

### 您的 API 密钥

- 您的 Gemini API 密钥使用 Chrome 存储 API 本地存储
- 除了 Google Gemini API，不会传输到任何其他服务器
- 您可以随时通过扩展设置删除它

## 数据存储

所有数据都存储在您的设备本地：

- **Chrome 存储**：API 密钥和用户偏好
- **本地存储**：摘要历史

您可以随时清除这些数据：
1. 卸载扩展
2. 清除 Chrome 的扩展数据
3. 使用扩展中的"清除历史"按钮

## 权限说明

QuickRead AI 请求以下权限：

- **activeTab**：读取当前网页内容
- **storage**：本地保存 API 密钥和偏好设置
- **scripting**：注入内容脚本以提取文本
- **sidePanel**：显示侧边栏界面
- **contextMenus**：添加右键菜单选项

这些权限仅用于扩展功能，不用于数据收集。

## 数据安全

- 与 Google Gemini API 的所有通信使用 HTTPS 加密
- 您的 API 密钥安全存储在 Chrome 的加密存储中
- 不会向第三方服务器传输数据

## 儿童隐私

QuickRead AI 不会故意收集 13 岁以下儿童的数据。本扩展面向普通用户。

## 政策变更

我们可能会不时更新此隐私政策。更改将发布在此页面，并更新"最后更新"日期。

您有权：
- 访问本地存储的数据
